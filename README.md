# 🌤️ Live Weather Dashboard

A fully responsive weather dashboard built with vanilla HTML, CSS, and JavaScript.

## Features (All 7 Requirements)

| # | Requirement | Status |
|---|-------------|--------|
| 1 | Geolocation API with fallback to Mumbai | ✅ |
| 2 | Fetch from Open-Meteo API (no key needed) | ✅ |
| 3 | Dynamic weather icons + gradient card background | ✅ |
| 4 | 7-day forecast grid with CSS-transition expand | ✅ |
| 5 | 24-hour Canvas chart (pure Canvas 2D, no libraries) | ✅ |
| 6 | City search with Geocoding API + debounce | ✅ |
| 7 | Responsive layout + debounced resize listener | ✅ |

## How to Run

Just open `index.html` in any browser. No server or dependencies needed.

## APIs Used

- **Weather**: `https://api.open-meteo.com/v1/forecast` (free, no API key)
- **Geocoding**: `https://geocoding-api.open-meteo.com/v1/search` (free, no API key)
